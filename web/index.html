$ npx anygent-mcp
How it works

4단계로 완성되는
나만의 에이전트

Plan → Research → Do → Check. 각 단계마다 Claude가 직접 물어보고, 검증하며 완성합니다.

🗺️
01 · plan

기획

에이전트의 목적, 대상, 도구, 금지사항을 인터뷰 형식으로 파악합니다.

🔍
02 · research

리서치

유사 에이전트와 모범 사례를 조사해 최적의 설계를 도출합니다.

⚙️
03 · do

생성

수집한 정보로 CLAUDE.md 설정 파일을 자동으로 작성합니다.

✅
04 · check

검수

샘플 3개를 실제 에이전트로 실행해 품질을 검증합니다.

MCP Tools

Claude Code에서
바로 사용하는 도구들

Anygent는 6개의 MCP 도구로 구성되어 있습니다. 순서대로 실행하면 에이전트가 완성됩니다.

anygent_plan 워크플로우 인터뷰 시작 및 기획
anygent_research 유사 에이전트 리서치 및 레퍼런스 수집
anygent_do CLAUDE.md 설정 파일 자동 생성
anygent_check 샘플 실행으로 품질 검증
list_templates 사용 가능한 템플릿 목록 조회
apply_template 템플릿 선택 및 커스터마이징
Installation

2분 만에 설치

npm 하나로 끝. Claude Code에 설정 한 줄 추가하면 바로 사용할 수 있습니다.

① 설치

# npm으로 설치
npm install anygent-mcp

# 또는 바로 실행
npx anygent-mcp

② Claude Code 설정

// ~/.claude/settings.json
{
  "mcpServers": {
    "anygent": {
      "command": "npx",
      "args": ["anygent-mcp"]
    }
  }
}
